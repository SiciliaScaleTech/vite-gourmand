# Vite & Gourmand - Plateforme de Commande

## Description
Application web de commande de repas à domicile pour l'entreprise "Vite & Gourmand".

## Installation en local
1. Cloner le projet : `git clone [URL-de-ton-repo]`
2. Importer les fichiers SQL (situés dans le dossier backend ou racine) :
   - Importer `structure.sql` (création des tables)
   - Importer `data.sql` (données de test)
3. Configurer le serveur local (WAMP/MAMP/XAMPP) sur le dossier `frontend`.

## Technologies utilisées
- PHP / MySQL
- HTML5 / CSS3
- Git (Git Flow)
